# Clone dulu Mas Mbak

### 1. Clone project

```bash

git clone https://github.com/RifqiArdian09/EssyCoff-Laravel.git

```

### 2. Copy file .env.example

```bash

copy .env.example .env

```
```bash
//samakan db
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=pos
DB_USERNAME=root
DB_PASSWORD=

```


### 3. Install dependency

```bash

composer install

```

### 4. Generate application key

```bash

php artisan key:generate

```

### 5. Munculkan gambar

```bash

php artisan storage:link

```
### 6. npm

```bash

npm install

```
### 7. Jalankan 
```bash

php artisan serve

```
```bash

npm run dev

```
